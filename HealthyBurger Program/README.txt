OOPTest HealthyBurger Excercise：

■Program Description:
This program is an implementation of a system for managing the selling of hamburgers in a fictitious company called "Bills Burgers". The system includes a class named "Hamburger" which represents a base hamburger, and two subclasses named "HealthyBurger" and "DeluxeBurger" which represent two popular variations of the base hamburger.

The "Hamburger" class has attributes to represent the name, meat, price, and bread roll type of a hamburger. It also has methods to add up to four additional items to a hamburger and calculate the total price of the hamburger. The "HealthyBurger" class inherits from the "Hamburger" class and adds two additional items that can be added to the healthy burger. The "DeluxeBurger" class also inherits from the "Hamburger" class and automatically adds chips and drinks as additional items to the deluxe burger.

The given code also includes a "Main" class that demonstrates how to use the three classes to create and customize hamburgers and calculate their prices. In the "Main" class, instances of the "Hamburger", "HealthyBurger", and "DeluxeBurger" classes are created and customized with additional items. The total price of each hamburger is then calculated and printed to the console.

Overall, the implementation demonstrates a simple example of object-oriented programming, including inheritance and method overriding.

■プログラム説明：
架空の会社である "Bills Burgers"でのハンバーガーの販売を管理するプログラムです。このシステムには、"Hamburger"という名前のクラスが含まれており、基本ハンバーガーを表し、"HealthyBurger"と"DeluxeBurger"という2つの人気のあるバリエーションを表す2つのサブクラスがあります。

"Hamburger"クラスには、ハンバーガーの名前、肉、価格、パンロールタイプを表す属性があります。また、最大4つの追加アイテムをハンバーガーに追加し、ハンバーガーの合計価格を計算するメソッドもあります。 "HealthyBurger"クラスは、"Hamburger"クラスを継承し、ヘルシーバーガーに追加できる2つの追加アイテムを追加します。 "DeluxeBurger"クラスも"Hamburger"クラスを継承し、デラックスバーガーに自動的にチップスとドリンクを追加します。

"Main"というクラスを含む、3つのクラスを使用してハンバーガーを作成し、カスタマイズし、価格を計算する方法を示しています。
"Main"クラスでは、"Hamburger"、"HealthyBurger"、および"DeluxeBurger"クラスのインスタンスが作成され、追加アイテムでカスタマイズされます。
その後、各ハンバーガーの合計価格が計算され、コンソールに出力されます。
全体として、継承やメソッドのオーバーライドを実装し、シンプルなオブジェクト指向を取り入れて作成しました。
